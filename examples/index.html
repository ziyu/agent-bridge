AgentBridge SDK Examples

Interactive demos showing how to use @agent-bridge/host and @agent-bridge/client to mount guest apps and establish bidirectional communication.

01 - Basic Inline Mode

Host mounts a guest app via raw code injection (srcdoc). Guest registers greet and add actions, host calls them. Demonstrates the simplest integration path.

Inline / Raw Host SDK Client SDK (IIFE)

02 - Iframe Mode

Host mounts a separate guest page via iframe URL. Guest provides getTime and toggleTheme actions. Shows how independently deployed apps communicate.

Iframe / URI Host SDK Client SDK (ESM)

03 - LLM Tool-Calling Integration

Full AI agent loop: guest registers weather capabilities → host converts to OpenAI/Anthropic/Gemini tool format → simulated LLM returns tool_calls → host routes actions back to guest.

LLM Integration Inline / Raw Host SDK
Run with pnpm run examples or npx serve . from the repo root.
